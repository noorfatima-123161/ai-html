## 2. product-page.html (Elegant Luxury Watch)

This file is the second task of Assignment 1 and represents an e-commerce product page designed using HTML with the assistance of AI prompt engineering.

### What it contains

The product page is divided into several structured sections:

* A navigation header containing menu items such as:

  * Shop
  * New Arrivals
  * Brands

* An image gallery section for product visuals

* A product details section containing:

  * Product title
  * Price
  * Description
  * Purchase buttons

This layout helped me understand how e-commerce websites organize product information in a user-friendly way.

### What I learned

From this task, I learned:

* Layout structuring in HTML
* Organizing sections for e-commerce pages
* Improving prompt-writing skills for AI-assisted development
* Understanding how AI can speed up front-end development while still requiring human review and correction

### Issues Faced

While working with AI-generated code, I noticed that some outdated HTML practices were included, such as:

* `<center>` tag
* `align` attribute
* `border` attribute

These deprecated elements are no longer considered best practice in modern HTML5. This assignment helped me recognize the importance of validating and optimizing AI-generated code according to current web standards.
